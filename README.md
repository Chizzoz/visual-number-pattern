A sleek, interactive web-based tool that visualizes numeric sequences as geometric patterns on a standard phone keypad.

[**Live Demo**](https://chizzoz.github.io/visual-number-pattern/)

---

## 🌟 Overview

**Visual Number Pattern** is a unique input method and visualization tool designed to transform simple numeric sequences into vibrant, color-coded patterns. By connecting numbers on a traditional 3x4 phone keypad, users can create intricate designs, making it a fun way to visualize passwords, PINs, or any number string.

## ✨ Key Features

- **Interactive Keypad**: Click or tap on numbers to build your sequence.
- **Dynamic Pattern Drawing**: Watch as the tool connects each number in real-time using a smooth canvas overlay.
- **Color-Coded Steps**: Each step in your sequence is assigned a unique color from a curated palette, making the order of input easy to follow.
- **Intelligent Repeat Handling**: Entering the same number multiple times creates concentric circles with increasing radii, adding depth to the visualization.
- **Dual Input Methods**: Use the on-screen keypad or type directly into the number display box.
- **Dark Mode Support**: Seamlessly switch between light and dark themes to suit your preference.
- **Fully Responsive**: Optimized for both desktop and mobile devices with intelligent canvas scaling.

## 🛠️ Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: Custom styling with a modern, clean aesthetic and CSS variables for theming.
- **JavaScript (Vanilla)**: Core logic for sequence tracking, theme switching, and interaction.
- **Canvas API**: High-performance rendering of the patterns and geometric shapes.

## 🚀 Getting Started

To run this project locally, simply:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Chizzoz/visual-number-pattern.git
    ```
2.  **Open the project**:
    Navigate to the project directory and open `index.html` in any modern web browser.

## 📖 How to Use

1.  **Click the Keypad**: Select numbers in the desired order. The pattern will automatically update.
2.  **Type Numbers**: Use the input box below the keypad to enter a sequence manually.
3.  **Visualize**: Observe the lines connecting your numbers and the concentric circles representing repeated digits.
4.  **Clear**: Hit the "Clear" button to reset the sequence and start fresh.
5.  **Toggle Theme**: Use the theme button at the top to switch between light and dark modes.
